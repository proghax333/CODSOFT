BlindReaper

Reaping the souls out of blind XSS vulnerable websites.

Detect and capture blind XSS attacks

BlindReaper provides a simple and intuitive user interface to manage your blind XSS targets and payloads. It also provides API to interact with the system, allowing third-party clients.

Manage multiple payloads simultaneously

BlindReaper is capable of managing multiple payloads at the same time, without interference. This simplifies the workflow and allows to manage threats from multiple domains, and separate concerns.
